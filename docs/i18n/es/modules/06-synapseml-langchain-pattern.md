# 06. Patrón de Integración con SynapseML y LangChain

## Patrón Práctico

1. Diseño de prompt y chain.
2. Ejecución distribuida de transformaciones.
3. Validación de salida y chequeos de calidad.
4. Handoff de salidas hacia etapa de pipeline Azure ML.

## Regla de Implementación

- No hardcodear secretos.
- Fijar versiones de dependencias.
- Mantener estable el esquema de entrada/salida.
